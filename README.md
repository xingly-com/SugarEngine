# 🚀方糖引擎（SugarEngine）
方糖引擎（SugarEngine）为QQ-NT机器人提供多维业务服务与用户管理能力的全新开放框架，支持组件插拔化安装等功能。

SugarEngine provides a new open framework for QQ-NT robots with multi-dimensional business services and user management capabilities, and supports plug-in and plug-in installation of components.
